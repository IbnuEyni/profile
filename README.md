## 🏗️ Featured Projects

### [Oracle Forge](https://github.com/IbnuEyni/oracle-forge)

Natural language data analytics agent evaluated on DataAgentBench (54 queries, 12 datasets, 9 domains). Uses a 3-layer knowledge base injection architecture to achieve 35.2% pass@1 against a 54.3% SOTA ceiling.

### [Tenacious Sales Bench](https://github.com/IbnuEyni/tenacious-sales-bench)

Domain-specific benchmark for B2B sales agents — 250 tasks across signal grounding, tone consistency, resource honesty, and workflow correctness. Published on HuggingFace with a SimPO-trained judge model.

### [Conversion Engine](https://github.com/IbnuEyni/Conversion-Engine)

Automated lead generation system with 5-signal enrichment pipeline (Crunchbase, job posts, layoffs, leadership changes, AI maturity), ICP classification, multi-channel outreach, and CRM sync.

### [The Ledger](https://github.com/IbnuEyni/10Acweek5)

Full CQRS/Event Sourcing stack for loan origination — PostgreSQL event store, cryptographic audit chain, MCP server (9 tools, 6 resources), counterfactual what-if analysis, regulatory package generation. 166 tests.

### [Automaton Auditor](https://github.com/IbnuEyni/10Acweek2)

Multi-agent "Digital Courtroom" for autonomous code auditing — 3 detective agents + 3 judicial agents + deterministic synthesis. LangGraph fan-out/fan-in parallelism. Audits a repo in 45s for $0.02.

### [Roo Code — Intent Traceability](https://github.com/IbnuEyni/Roo-Code)

Intent-Code Traceability System for VS Code — middleware hooks that force AI agents to declare intent before writing code. Two-stage state machine, SHA-256 content hashing, multi-agent coordination via shared brain.
